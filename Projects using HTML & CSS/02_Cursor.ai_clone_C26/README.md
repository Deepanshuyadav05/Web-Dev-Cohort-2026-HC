## üé® Design Specifications

### Fonts
The project utilizes a clean, CursorGothic, sans-serif typography stack to mimic the **Inter** and **System UI** aesthetic of the original site.
[Find font here](https://github.com/imjac0b/CursorGothic) \
***How to use this font (as it is not officialy availabe)***
* make a folder in root directory of name fonts 
* In that file add this: \
 @font-face { \
    font-family: CursorGothic; \
    src: url('./fonts/CursorGothic-Regular.woff2') format('woff2'); \
    font-display: swap; \
    font-weight: 400; \
   }  
* now simply add the font where u want to add in the css file : \
font-family: CursorGothic, sans-serif;

* **Headings:** Bold weights with tight letter-spacing.
* **Body:** High-legibility spacing with a focus on a "Dark Mode" aesthetic.
